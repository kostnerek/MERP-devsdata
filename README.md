# MERP-devsdata
### Running
You need to have python3 with pip, node and npm installed on your computer.<br>
Run those commands to start app<br>
`$ pip install -r requirements`<br>
`$ python3 app.py`<br>
`$ python3 create_example_events.py`<br>
`$ npm i`<br>
`$ npm run start`<br>

Now go on `localhost:5000/`
